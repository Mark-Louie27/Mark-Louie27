## 🚀 Featured Projects

### 🧾 Singil — Invoice Automation Platform
> Modern billing and invoicing built for Filipino freelancers and small businesses.

- 📄 PDF invoice generation with client management
- 💳 PayMongo payment integration
- 🏛️ BIR-ready invoice workflow
- 📧 Automated email delivery via Resend
- 🔐 Auth via Clerk, storage via Supabase

**Stack:** `Next.js 15` `TypeScript` `Prisma` `PostgreSQL` `Supabase` `Clerk` `Resend`

---

### 💰 FineTrack — Personal Finance Tracker
> A budgeting and expense tracking app built for the Philippine market.

- 🏦 Wallet CSV imports (GCash, Maya, UnionBank)
- 📊 Financial analytics dashboard
- 💳 Subscription plan gating system
- 🔍 SEO infrastructure
- ⚡ Built with TanStack Start (full-stack React)

**Stack:** `TanStack Start` `TypeScript` `Neon PostgreSQL` `Stripe` `Plaid`

---

### 🏫 Automated Classroom Scheduling System (ACSS)
> A scheduling system for academic institutions using constraint-based algorithms.

- 🗓️ Automated schedule generation with conflict detection
- 🏗️ MVC architecture with repository/service pattern
- 📋 Curriculum and section management
- 🔒 Role-based access (Chair, Admin)

**Stack:** `PHP` `MySQL` `MVC` `Vanilla JS`
